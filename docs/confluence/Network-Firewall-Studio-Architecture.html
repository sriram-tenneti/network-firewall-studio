Part 1: C4 Architecture Model

1. Executive Summary

Network Firewall Studio is an enterprise-grade web application designed to modernize and streamline the lifecycle management of network firewall rules across NGDC (Next-Generation Data Center) infrastructure.

Key Capabilities

CapabilityDescription
New Rule DesignDrag-and-drop interface for creating firewall rules with strict naming standards enforcement
Migration ManagementAutomated migration of legacy firewall rules from Legacy Data Centers to modernized NGDC infrastructure
Standards EnforcementComprehensive naming standards engine with group-to-group policy enforcement
Rule CertificationPeriodic certification workflows to ensure compliance and recertification
Change ManagementIntegration with ServiceNow for CHG request submission and tracking
GitOps IntegrationRepository-based rule storage and version control

7. Level 4: Code - Naming Standards Engine

Core Module: app/services/naming_standards.py
The Naming Standards Engine is the heart of standards enforcement, providing validation, generation, and suggestion capabilities for all firewall object names.

Naming Standards Patterns

TypePrefixPatternExample
Groupgrp-grp-{AppID}-{NH}-{SZ}-{Subtype}grp-APP001-NH01-CDE-WEB
Serversvr-svr-{AppID}-{NH}-{SZ}-{ServerName}svr-APP001-NH05-GEN-DBPROD01
Subnetrng-rng-{AppID}-{NH}-{SZ}-{Descriptor}rng-APP001-NH12-DMZ-WEBSUBNET

Valid Subtypes

SubtypeDescriptionSubtypeDescription
WEBWeb serversCACHECache servers
APPApplication serversLBLoad balancers
DBDatabase serversPROXYProxy servers
BATCHBatch processingMONMonitoring servers
MQMessage queue serversAPIAPI gateway servers

Validation Rules

RuleDescriptionError Message
Prefix CheckName must start with grp-, svr-, or rng-"Invalid prefix: must be grp-, svr-, or rng-"
Segment CountMust have exactly 5 segments"Invalid format: expected 5 segments"
AppID ValidAppID must exist in the application registry"Unknown AppID: {value}"
NH ValidNH must be NH01 through NH17"Invalid Neighbourhood: {value}"
SZ ValidSZ must be a recognized security zone code"Invalid Security Zone: {value}"
Case CheckAll segments must be uppercase"Name must be uppercase"
